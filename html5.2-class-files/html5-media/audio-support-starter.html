New Media Elements: Audio

Elements:

<video>
<audio>

Audio Attributes:

Attribute Value What id does
autoplay autoplay Plays the audio automatically
controls controls Displays controls: play, pause, volume
loop loop Plays the audio over again after its finished playing
preload preload Loads the audio at page load, ignored if "autoplay" is present
src [url] Sets url of the audio to play

Checking for Browser Support

To check for browser support of <video> and <audio> elements you can use JavaScript to:

Dynamically create the <video> and <audio> elements.

Check for the existence of a method on the element.

If the method doesn't exist the browser does not support the element.

i.e.:

var hasAudio = (document.createElement('audio').canPlayType);
