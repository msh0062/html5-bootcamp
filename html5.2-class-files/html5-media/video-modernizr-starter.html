New Media Elements: Video

Elements:

<video>
<audio>

Video Attributes:

Attribute Value What it does
audio muted Sets the default "state". Only muted is supported
autoplay autoplay Plays the video automatically
controls controls Displays controls: play, pause
height [pixel value] Sets the height of the video player
loop loop Plays the video over again after its finished playing
poster [url] Displays an image in place of the video, until the video is played
preload preload Loads the video at page load, ignored if "autoplay" is present
src [url] Sets url of the video to play
width [pixel value] Sets the width of the video player

Checking for Browser Support

To check for browser support of <video> and <audio> elements you can use JavaScript to:

Dynamically create the <video> and <audio> elements.

Check for the existence of a method on the element.

If the method doesn't exist the browser does not support the element.

var hasVideo = (document.createElement('video').canPlayType);
