After the page opens, but before clicking the "Open Dialog" button:

  1. Move the slider bar below.
  2. Click the "Open Dialog" button.
  3. Move the slider bar below.
  4. Close the dialog box.
  5. Move the slider bar below.

The dialog box is a modal dialog box, which means the user must interact with the dialog box before it can continue interacting with the application. The modal box prevents any interactions from occuring on the current page that spawned the modal box.
